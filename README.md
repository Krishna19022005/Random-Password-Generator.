# Random-Password-Generator.
A simple and secure Random Password Generator built using HTML, CSS, and JavaScript. This tool allows users to generate strong passwords instantly based on customizable preferences.

**Features**
Generate random passwords instantly
Choose password length, 
Include or exclude: 
Uppercase letters (A-Z), 
Lowercase letters (a-z), 
Numbers (0-9), 
Symbols (@, #, $, etc.), 
One-click Copy to Clipboard button, 
Clean and user-friendly UI, 
Fully responsive design, 

**Technologies Used**
HTML – Structure of the app, 
CSS – Styling and layout, 
JavaScript – Password generation logic. 

 **How to Use**
Open the app in your browser.
Select your desired password length.
Check the character types you want to include.
Click Generate Password.
Click Copy to copy the password.
